spring boot开发天猫

spring boot实现天猫前端和后台管理

选用技术

  1. Java
  2. 前端：html, CSS, Javascript, JSON, AJAX, JQuery ,Bootstrap, Vue.js
  3. 框架：spring springMVC spring boot
  4. 中间件： redis,nginx,elasticsearch,shiro
  5. 数据库：MySQL
  6. 开发工具：idea,maven

需求分析-前台

  1. 首页
    （1）在横向导航栏上提供4个分类连接
    （2）在纵向导航栏上提供全部17个分类连接
    （3）当鼠标移动到某一个纵向分类连接的时候，显示这个分类下的推荐商品
    （4）按照每种分类显示5个商品的方式，显示所有17种分类

  2. 分类页
    （1）显示分辨率为950x100的当前分类图片
    （2）显示本分类下的所有产品

  3. 查询结果页
    显示满足查询条件的商品

  4. 产品页
    （1）显示分辨率为950x100的当前商品对应的分类图片
    （2）显示本商品的5个单独图片
    （3）商品的基本信息，如标题，小标题，价格，销量，评价数量，库存等
    （4）商品详情
    （5）评价信息
    （6）5张商品详细图片

  5. 结算页面
    在结算页面显示被选中的订单项

  6. 支付页面
    确认支付页面显示本次订单的金额总数

  7. 支付成功页面
    付款成功时，显示本次付款金额

  8. 购物车页面
    在购物车中显示订单项

  9. 我的订单页面
    显示所有订单，以及对应的订单项

  10. 确认收货页面
    （1）显示订单项内容
    （2）显示订单信息，收货人地址等

  11. 确认收货成功页面

  12. 进行评论页面
    （1）显示要评价的商品信息，商品当前的总评价数
    （2）评价成功后，显示当前商品所有的评价信息

  13. 登陆页面

  14. 注册页面

需求分析后台-后台

  1. 分类管理
  2. 分类属性管理
  3. 产品管理
  4. 产品属性设置
  5. 产品图片管理
  6. 用户管理
  7. 订单管理
